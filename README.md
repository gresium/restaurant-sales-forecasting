# restaurant-sales-forecasting

This repository contains an end-to-end machine learning project that predicts
future restaurant demand based on historical sales data.

The project includes preprocessing, exploratory data analysis (EDA),
feature engineering, machine learning models (Linear Regression, Random Forest, CatBoost),
and a 12-month forecasting pipeline.
